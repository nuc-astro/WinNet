109  end do

145  end do

525  end if

527  end do
528

804 !>
805 !! Reads the trajectory file.
806 !!
807 !! The columns and structure of the file is determined by the user
808 !! defined parameter \ref parameter_class::trajectory_format.
809 !! The default trajectory format is given by: <b> "time temp dens rad ye" </b>.
810 !! This format will assume a file that looks like:
811 !! \file{
812 !! \#time[s], T9[GK], density[g/cm^3], R[km], Ye
813 !! \#--------------------------------------------
814 !! 0.0000e+00 1.0965e+01 8.7096e+12 4.9273e+01 0.03488
815 !! 2.5000e-05 9.9312e+00 7.4131e+12 5.0361e+01 0.03488
816 !! ...}
817 !! A more complex format, <b> "time:ms x y z log_dens log_temp:K ye" </b>, would assume a file like:
818 !! \file{
819 !! \#t [ms] x[km] y[km] z[km] log(rho[cgs]) log(T[K]) Ye
820 !! \#----------------------------------------------------------------------------------
821 !! 0.0000E+00 0.1051E+02 0.3774E+01 -0.4538E+01 0.1364E+02 0.1016E+02 0.1604E-01
822 !! 0.2521E-01 0.1046E+02 0.4724E+01 -0.4541E+01 0.1363E+02 0.1060E+02 0.1604E-01
823 !! ...}
824 !! Neutrino luminosities and energies are also read here. A format could look like:
825 !! <b> "time temp dens rad ye lnue lanue tnue tanue" </b>. \n
826 !! Possible column names are:
827 !! <pre style="line-height:.3">
828 !! - time : The time (s) of the trajectory
829 !! - temp : The temperature (GK) of the trajectory
830 !! - dens : The density (g/ccm) of the trajectory
831 !! - rad : The radius (km) of the trajectory
832 !! - x,y,z : x, y, and z coordinates (km) of the trajectory
833 !! - ye : The electron fraction
834 !! - lanue : Anti-electron neutrino luminosities (erg/s)
835 !! - lnue : Electron neutrino luminosities (erg/s)
836 !! - tanue : Anti-electron neutrino temperatures (MeV)
837 !! - tnue : Electron neutrino temperatures (MeV)
838 !! - eanue : Anti-electron neutrino energies (MeV)
839 !! - enue : Electron neutrino energies (MeV)
840 !! - lanux : Anti-neutrino luminosities of heavy neutrinos (erg/s)
841 !! - lnux : Neutrino luminosities of heavy neutrinos (erg/s)
842 !! - tanux : Anti-neutrino temperatures of heavy neutrinos (MeV)
843 !! - tnux : Neutrino temperatures of heavy neutrinos (MeV)
844 !! - eanux : Anti-neutrino energies of heavy neutrinos (MeV)
845 !! - enux : Neutrino energies of heavy neutrinos (MeV)
846 !! - skip : skip a column
847 !! .
848 !! </pre>
849 !!
850 !! @note Lines starting with "#" or blank lines are skipped
851 !!
852 !! @author Moritz Reichert
853 !!
854 !! \b Created: 17.12.20
